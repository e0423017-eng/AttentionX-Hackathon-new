# AttentionX — Automated Content Repurposing Engine

Turn long-form videos into short, viral-ready clips with AI-powered peak detection, smart vertical cropping, and dynamic captions.

## Live Demo
[Open the app here](https://automated-content-re-3clx.bolt.host/)


## What it does
AttentionX helps mentors, educators, and creators convert long videos into snackable short clips for Reels, Shorts, and TikTok.

## Features
- Emotional peak detection.
- Auto vertical 9:16 smart crop.
- Dynamic captions and hook generation.
- Clean hackathon-friendly UI.
- Fast preview and shareable output.

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Supabase

## How to run locally
```bash
npm install
npm run dev
```

## How to use
1. Open the app.
2. Paste a video URL or upload a file.
3. Let the app process the video.
4. Download or preview the generated clip.

## Project Structure
- `src/` — main source code.
- `supabase/` — database/migrations.
- `index.html` — app entry.
- `vite.config.ts` — Vite config.
